# FlyAndEarn Business Model

## Revenue Streams

### 1. Platform Commission (Primary Revenue)

**Recommended: 10% of total transaction value**

| Component | Rate | Example (€200 product + €30 service fee) |
|-----------|------|------------------------------------------|
| Product Value | - | €200 |
| Service Fee (to traveler) | max 15% | €30 |
| **Platform Fee** | **10% of total** | **€23** |
| **Buyer Pays** | | **€253** |

**Breakdown:**
- Traveler receives: €200 (product) + €30 (service fee) = €230
- Platform receives: €23 (10% of €230)

### 2. Alternative: Dual-Sided Commission

Charge both parties for fairness:

| Party | Fee | Example |
|-------|-----|---------|
| Buyer | 5% of total | €11.50 |
| Traveler | 5% of service fee | €1.50 |
| **Total Platform Revenue** | | **€13.00** |

#### Buyer Plus (€4.99/month)

## Unit Economics

### Per Transaction (€230 average value)

**Costs:**
| Item | Cost |
|------|------|
| Payment processing (Stripe) | 1.4% + €0.25 = €3.47 |
| Fraud prevention | €0.50 |
| Customer support (allocated) | €1.00 |
| Infrastructure | €0.20 |
| **Total Cost** | **€5.17** |

**Revenue (10% commission):**
| Item | Amount |
|------|--------|
| Platform fee | €23.00 |
| Less: Costs | -€5.17 |
| **Net Margin** | **€17.83 (77.5%)** |

### Break-Even Analysis

| Monthly Costs | Amount |
|---------------|--------|
| Servers & Infrastructure | €500 |
| Payment gateway fixed | €50 |
| Customer support (1 person) | €2,500 |
| Marketing | €1,000 |
| Legal & Compliance | €500 |
| Misc | €450 |
| **Total Monthly Costs** | **€5,000** |

**Transactions needed to break even:**
- €5,000 ÷ €17.83 = **281 transactions/month**
- At €230 avg value = **€64,630 GMV/month**
